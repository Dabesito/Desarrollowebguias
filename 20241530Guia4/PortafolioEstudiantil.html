Descripción del curso

El curso de Desarrollo Web busca que el estudiante adquiera habilidades prácticas en la creación de sitios web responsivos utilizando HTML, CSS y JavaScript. Se abordan temas como estructura semántica, diseño adaptable, frameworks modernos y optimización web.

Contenidos principales:

  • Introducción al HTML5 y estructura semántica
  • Diseño adaptable con Flexbox y Grid
  • Uso de CSS avanzado y Media Queries
  • Introducción a JavaScript y DOM
